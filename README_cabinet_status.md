# 柜子状态看板（PyQt）

本程序用于显示 **更衣柜/更鞋柜** 与 **发鞋柜** 的门状态。

## 运行方式（源码）

1. 安装依赖：

```bat
pip install -r requirements.txt
```

2. 配置数据库连接：

编辑同目录下 `db_config.ini`（与人员注册程序一致）。

发鞋柜的男/女分组会**自动**根据 `OperRoom.dbo.Device.Name`（男发鞋柜/女发鞋柜）识别。
（仍保留 `db_config.ini` 的 `[disshoe]` 作为兜底，但一般不需要配置。）

3. 启动：

```bat
python cabinet_status_main.py
```

## 使用说明

- 顶部选择 **柜子类型**（更衣/更鞋柜 或 发鞋柜）
- 更衣/更鞋柜：选择具体柜子后，界面会展示该柜全部门（默认每列 12 个），点击门在右侧查看详情。
- 发鞋柜：一次性显示所有柜子（按男发鞋柜/女发鞋柜分 Tab），不显示右侧详情。
  - **占用/空** 与 **锁定/未锁定** 都通过门颜色区分
  - 占用时会将姓名直接显示在门上
  - 若该门有鞋信息（Size/Style 任一不为空），门上会显示一张拖鞋图片

拖鞋图片说明：程序会在启动时生成/覆盖 `assets/slipper.png`（离线可用）。
如果你希望显示你们现场的“真实拖鞋照片”，直接用同名文件替换该 PNG 即可。

## 打包 EXE（可选）

运行：

```bat
build_cabinet_status_exe.bat
```

生成的 EXE 在 `dist/` 目录。
